STOCK MANAGEMENT SYSTEM

Stock Management System is a full-stack Java-based web application for managing stocks, customers and user portfolios. Itâ€™s designed for both administrators and end customers, with secure authentication and REST back-end APIs.

Features

  • Authentication: Login and sign-up for users.
  • Stocks Management: Add, update, delete, and list stocks
  • Customer Management: Admin can add and remove customers.
  • Buy/Sell Transactions: Customers can purchase or sell stocks, with changes reflected in their portfolios.
  • User Portfolio: Real-time holdings, market value, and profit/loss summary per customer.
  • Role-based Access: Distinct functionality for admins vs. end customers.

    • Tech Stack

    • Backend: Java, Spring Boot, Spring JDBC
    • Database: Oracle 19c
    • Frontend: OJET
    • Build Tool: Maven
    • REST APIs: For all features and reporting
    • Security: Basic authentication, validation
